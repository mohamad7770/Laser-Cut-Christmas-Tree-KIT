 Laser-Cut Christmas Tree Project

Digital manufacturing project demonstrating complete workflow from design to assembly using laser cutting technology.

 About
- Objective: Transform digital design into tangible product
- Technology: Laser cutting
- Material: 3mm plywood
- Software: CorelDRAW, CAD

 Workflow
1. Design: Create model and export as DXF
2. Preparation: Prepare file in CorelDRAW
3. Cutting: Execute laser cutting
4. Assembly: Assemble final pieces
